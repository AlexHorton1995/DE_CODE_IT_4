Basic SQL Questions:

1. What does SQL stand for? Structured Query Language
2. What is SQL used for? SQL is used to communicate with a database.
3. What does the acronym RDBMS stand for? Relational Database Management System
4. Name 3 different RDBMS systems. SQL, Oracle, MySQL
5. What does the acronym C.R.U.D stand for? Create, Read, Update, Delete
