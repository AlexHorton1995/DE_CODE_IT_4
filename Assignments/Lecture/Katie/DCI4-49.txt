
Basic SQL Questions:

1. What does SQL stand for?  Structured Query Language
2. What is SQL used for? Language that is used to communicate with a database to store, retrieve, manage, and manipulate data within a database. 
3. What does the acronym RDBMS stand for? Relational database management system 
4. Name 3 different RDBMS systems. Oracle, MySQL, Microsoft SQL server
5. What does the acronym C.R.U.D stand for? Create, Read, update, delete
